# КиноПоиск — Смотреть фильм

Расширение для Firefox и Chrome: на сайте [kinopoisk.ru](https://www.kinopoisk.ru/) появляется кнопка **«Смотреть фильм»**. По нажатию открывается новая вкладка с [sspoisk.ru](https://www.sspoisk.ru).

## Установка

### Chrome

1. Откройте Chrome и перейдите в `chrome://extensions/`
2. Включите **«Режим разработчика»** (переключатель справа вверху)
3. Нажмите **«Загрузить распакованное расширение»**
4. Укажите папку `kinoext` (где лежат `manifest.json`, `content.js`, `content.css`)

### Firefox

1. Откройте Firefox и перейдите в `about:debugging#/runtime/this-firefox`
2. Нажмите **«Загрузить временное дополнение...»**
3. Выберите файл `manifest.json` из папки `kinoext`

После установки зайдите на любой фильм или сериал (https://www.kinopoisk.ru/series/257376/) КиноПоиска — в правом нижнем углу будет оранжевая кнопка «Смотреть фильм/сериал».
## Файлы

- **manifest.json** — описание расширения и правила запуска скрипта на kinopoisk.ru
- **content.js** — добавляет кнопку и по клику открывает sspoisk.ru в новой вкладке
- **content.css** — стили кнопки (фиксированная позиция, цвет, тень)
